Intro to Day 02 Notes

  • Most code in this 30 Days of React will be in ES6
  • ES5 was finalized in 2009
  • ES6 was finalized in 2015 (but IE11 does not support it)
  • JSX is a preprocessor step that adds XML syntax to JavaScript. You can use React without JSX, but it makes React a lot more elegant.

In order to reap the benefits of ES6, we must -

  • Transpile the code so that a wider range of browsers understand our JavaScript. This converts ES6 into ES5.
  • Have to include a shim or polyfill that provides functionality added in ES6 that a browser may or may not have.
